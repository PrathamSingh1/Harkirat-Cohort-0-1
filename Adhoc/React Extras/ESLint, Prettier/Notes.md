What is Linting -> 

Linting is the process of running a program that analyzes source code to flag programming errors, bugs, stylistic errors.

Why lint ->

1. Multiple develpers in a team can have a similar coding pattern
2. Good readability of the code 
3. Authomatic Linting so developer doesn't have to manually lint their code

Tools ->

1. ESLint - https://eslint.org/
2. Prettier - https://prettier.io/
3. Husky  - https://typicode.github.io/husky/
